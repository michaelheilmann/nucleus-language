# `DL_StringBuffer_getSize`
*Get the size of a string buffer.*

## C Specification
```
size_t
DL_StringBuffer_getSize
    (
        DL_Context *context,
        DL_StringBuffer *stringBuffer
    );
```

## Parameters
- `context`: a pointer to the `DL_Context` object
- `stringBuffer`: a pointer to the DL_StringBuffer` object`
